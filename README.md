# Quiz Game Application using C++

A console-based quiz game developed in C++ that allows users to practice questions from multiple topics and difficulty levels.

## Features
- Multiple topics (C++, Python, JavaScript, HTML & CSS, Maths, Tech, GK) (more can be added later)
- Three difficulty levels (Easy, Medium, Hard)
- File-based question loading
- Random question selection
- User-defined number of questions per quiz
- Instant feedback with scoring
- Negative marking for wrong answers

## Folder Structure
├── src/main.cpp
├── questions/
│ ├── cpp.txt
│ ├── python.txt
│ ├── javascript.txt
│ ├── html_css.txt
│ └── ..



## How to Run
clang++ src/main.cpp -o quiz
./quiz

##Scoring System
Correct answer: +1
Wrong answer: −0.25
Skipped question: 0
